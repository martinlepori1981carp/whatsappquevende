# Página de Agradecimiento - WhatsApp que vende solo

Esta es la página de agradecimiento post-compra configurada para tu curso.

## Cómo subir a Vercel

1.  Asegúrate de tener instalada la CLI de Vercel o usa el dashboard web.
2.  Sube el contenido de esta carpeta (o el archivo ZIP adjunto) a tu proyecto en Vercel.
3.  Vercel detectará automáticamente el archivo `vercel.json` y configurará la ruta.

## Información del Proyecto

-   **URL de la página:** `https://whatsappquevende.vercel.app/gracias`
-   **Acciones incluidas:**
    -   Botón de acceso al curso.
    -   Botón de descarga automática del PDF.
    -   Botón de contacto directo por WhatsApp con mensaje pre-armado.
